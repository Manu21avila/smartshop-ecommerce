# 🛍️ SmartShop - Mini E-commerce com Dashboard Administrativo

Projeto completo de e-commerce desenvolvido para simular um sistema real de vendas online, incluindo carrinho persistente, controle de produtos e painel administrativo com indicadores de desempenho.

---

## 🎯 Objetivo do Projeto

Desenvolver um mini sistema de loja virtual com foco em:

- Experiência do usuário
- Regra de negócio
- Manipulação de dados
- Dashboard analítico
- Organização de código modular

---

## 🧠 Funcionalidades

### 🛒 Loja
- Listagem dinâmica de produtos via JSON
- Filtro por categoria
- Busca por nome
- Ordenação por preço
- Badge de promoção
- Controle de estoque visual
- Carrinho persistente com localStorage

### 🧾 Carrinho
- Listagem de produtos adicionados
- Remoção de itens
- Cálculo automático de subtotal
- Frete fixo
- Aplicação de cupom de desconto

### 📊 Painel Administrativo
- Receita total
- Total de itens vendidos
- Produto mais vendido
- Gráfico de vendas (Chart.js)

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
- JSON para simulação de banco de dados
- Chart.js para visualização de dados

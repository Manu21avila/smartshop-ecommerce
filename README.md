# SmartShop - Projeto de E-commerce

Projeto de e-commerce desenvolvido com JavaScript puro para demonstração de habilidades front-end.

## Funcionalidades

- Listagem dinâmica de produtos
- Carrinho com LocalStorage
- Busca de produtos
- Cupom de desconto
- Cálculo de frete
- Layout responsivo

## Tecnologias

- HTML
- CSS
- JavaScript
